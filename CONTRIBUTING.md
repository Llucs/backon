# Contributing to backon

Thanks for considering contributing to backon!

## Setup

```bash
git clone https://github.com/Llucs/backon.git
cd backon
pip install pdm
pdm install
```

## Contribution workflow

1. Fork the repository.
2. Create a feature branch from `main`.
3. Make your changes.
4. Run the checks described in [`AGENTS.md`](AGENTS.md) (ruff, mypy, pytest).
5. Commit with a descriptive message referencing any relevant issues.
6. Push your branch and open a pull request against `main`.

## Project rules

All development rules — code style, architecture, testing, versioning, release process, and pre-commit checks — are documented in [`AGENTS.md`](AGENTS.md). Read it before making changes.
